# Habit Streak Tracker

A web-based application designed to help users track their daily habits, maintain streaks, and stay motivated with dynamic content like quotes, affirmations, and fun facts. Built with a modern, responsive UI, the app includes persistent history tracking, toggleable sections, and a visually appealing layout with smooth transitions.

## Features

- **7-Day History Section**: View your habit streak history for the past 7 days with a toggle to show/hide.
- **Daily Fun Facts and Quotes**: Rotates daily fun facts and motivational quotes to keep users engaged.
- **Proper Date Display**: Displays the current date for daily content, ensuring timely and relevant updates.
- **Toggle Buttons**: Easily show or hide history and suggestion sections for a personalized experience.
- **Visual Hierarchy**: Improved layout with consistent icon usage, proper spacing, and clear section organization.
- **Persistent History**: Stores the last 7 days of habit data locally to track progress over time.
- **Responsive Design**: Adapts seamlessly to different screen sizes for a consistent experience on mobile and desktop.
- **Additional Features**:
  - Background music for a calming user experience.
  - Daily affirmations to boost motivation.
  - Smooth transitions for a polished interface.

## UI Highlights

- **Clear Section Organization**: Intuitive layout with distinct sections for habits, history, quotes, and more.
- **Consistent Icon Usage**: Icons enhance usability and provide visual cues for actions and sections.
- **Better Spacing and Visual Hierarchy**: Clean design with ample spacing for readability and focus.
- **Smooth Transitions**: Animations for toggling sections and updating content for a fluid experience.
- **Date-Based Content Rotation**: Fun facts, quotes, and affirmations update daily based on the current date.
- **Toggle Controls**: Enable or disable optional sections like history or suggestions with ease.
- **Persistent History Tracking**: Uses local storage to maintain a 7-day history of habit completion.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mentalhealthadvocacynow/gentle-penguin-rest.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gentle-penguin-rest
   ```
3. Open `gentle-penguin-rest` on vercel in a web browser to run the app.

## Usage

1. **Track Habits**: Mark daily habits as completed to build and maintain streaks.
2. **View History**: Toggle the 7-day history section to review your progress.
3. **Explore Daily Content**: Check daily fun facts, quotes, and affirmations for motivation.
4. **Customize View**: Use toggle buttons to show or hide history and suggestion sections.
5. **Responsive Experience**: Use the app on any device, with a layout that adapts to your screen size.

## Dependencies

- No external dependencies required for core functionality. The app uses vanilla HTML, CSS, and JavaScript.
- Optional: CDN-hosted libraries (e.g., Font Awesome for icons) may be included for styling.
